# 🔒 VaultChat

A secure, encrypted messaging dashboard with a beautiful dark theme.

## ✨ Features

- 🔐 **End-to-End Encryption Active** - All messages are locally encrypted
- 💬 **Interactive Chats** - Click any contact to open full conversation
- 👥 **Group Chat Support** - See messages from team members
- 🔒 **Secret Chat** - Self-destructing messages for extra privacy
- 📱 **Mobile-Optimized** - Works perfectly on phones
- 🌙 **Modern Dark Design** - Easy on the eyes
- 🎯 **Unread Badges** - Know when you have new messages
- ⚡ **Fast & Responsive** - Built with Tailwind CSS

## 🛠️ Technologies Used

- **HTML5** - Structure
- **CSS (Tailwind CSS)** - Styling
- **JavaScript** - Interactivity
- **Google Fonts** - Hanken Grotesk, Inter

## 🚀 Live Demo

[View VaultChat Live](https://lumenn-knight00.github.io/vaultchat/)

## 📱 Install as App

1. Open the live demo on your phone
2. Tap the browser menu (three dots)
3. Select **"Add to Home Screen"**
4. Tap **"Install"**
5. Enjoy VaultChat as a native app!
